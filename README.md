# 🎉 Birthday & Proposal Website

A beautiful interactive web project combining:

- 🎂 Birthday celebration page
- 💖 Proposal flow with multiple questions
- 💌 Love letter popup
- 💬 Message saving system (local + server)
- 🎡 3D image carousel
- ✨ Animations (hearts, butterflies, fireworks)

---

## 🚀 Features

### 🎂 Birthday Page
- Countdown timer
- 3D rotating image carousel
- Surprise gift section with video
- Floating hearts & butterflies effects
- Krishna image animation
- Clickable envelope with love letter popup 💌

### 💖 Proposal System
- Multi-step proposal questions (7 steps)
- "No" moves to next question
- Final "No" button runs away 😄
- "Yes" redirects to celebration page

### 💬 Message System
- Users can write messages
- Messages saved:
  - Locally (browser)
  - OR to server (`messages.txt`)

## ⚙️ Setup Instructions

### 1️⃣ Install Node.js

### 2️⃣ Install Dependencies

npm install express  

### 3️⃣ Run Server

node server.js  

### 4️⃣ Open Website

http://localhost:3000
